# Code of Conduct

The `twentytwohundred` org adopts the **[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** as its Code of Conduct. The full text is canonical at the link above; this file commits the org to it.

## Reporting

Concerns about behavior in any 2200 project space (issues, pull requests, discussions, code review, communications via project channels) can be reported privately to:

**Email:** [hello@2200.ai](mailto:hello@2200.ai)

Reports are reviewed promptly and confidentially. Reasonable steps are taken to respect the privacy of the reporter.

## Scope

This Code of Conduct applies to all spaces operated by the `twentytwohundred` org: GitHub repositories, the wiki, GitHub Discussions, and any project communication channel where someone is acting as a representative of a 2200 project.

## Enforcement

The org's standing posture follows the Contributor Covenant 2.1 enforcement guidelines. Org administrators have discretion to apply consequences proportional to the impact of the conduct, ranging from a private warning to a permanent ban from the project's spaces.

## Per-repo overrides

Individual repos may add a `CODE_OF_CONDUCT.md` if they need project-specific scope. When they do, the per-repo file takes precedence for that project; the org-wide file applies to all other projects.
